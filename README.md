#  TSP Visualizer

##  Описание проекта
**TSP Visualizer** — это интерактивное веб-приложение для наглядной демонстрации алгоритмов решения задачи коммивояжёра (Traveling Salesman Problem). 
##  Как работает
Пользователь может расставлять города на карте, выбирать один из трёх алгоритмов поиска маршрута и наблюдать за пошаговой анимацией его построения. 

##  Основные возможности
-  **Интерактивная карта** — добавление городов кликом мыши на поле Canvas.
-  **Три алгоритма поиска маршрута:**
  - *Жадный* — на каждом шаге переходит в ближайший непосещённый город. Очень быстрый, но часто находит далёкий от оптимального маршрут.
  - *Полный перебор* — перебирает все возможные маршруты и выбирает самый короткий. Точное решение только для ≤ 10 городов, так как количество вариантов растёт факториально.
  - *Муравьиный* — имитирует колонию муравьёв. «Муравьи» прокладывают маршруты и оставляют феромон на коротких путях. Со временем все муравьи идут по самому «пахучему» (короткому) маршруту.
  - *Ветви и границы* — перебирает маршруты, «умно» — отсекает (не рассматривает) те ветки, которые заведомо хуже уже найденного лучшего маршрута.
-  **Пошаговая анимация** — визуализация процесса построения маршрута.
-  **Статистика в реальном времени** — длина маршрута, время выполнения алгоритма, количество городов.
-  **Лог анимации** — отображение каждого шага построения маршрута.

##  Технологический стек

| Характеристика |  Frontend  |  Backend  |
| :--- | :--- | :--- |
| **Технологии** | JavaScript (ES6+), Canvas API, CSS3 | Python 3.12, FastAPI, Pydantic |
| **Основная роль** | Интерфейс, отрисовка карты, анимация | Вычисление тяжелых алгоритмов (TSP) |
| **Где выполняется**| В браузере пользователя (клиент) | Внутри Docker-контейнера (сервер) |
| **Взаимодействие** | Отправляет координаты городов через fetch | Принимает JSON, возвращает кратчайший путь |

 ##  Запуск проекта

### Способ 1: Через Docker

```bash
git clone https://github.com/dasanauhackaa-cell/tsp-visualizer.git
cd tsp-visualizer
docker compose up --build
```
### Способ 2:
Терминал 1:
```bash
git clone https://github.com/dasanauhackaa-cell/tsp-visualizer.git
cd tsp-visualizer
python -m venv .venv
# Windows:
.venv/Scripts/activate
# Linux:
source .venv/bin/activate
pip install -r requirements.txt
cd backend
uvicorn app.main:app --reload --port 8000
```
Терминал 2:
```bash
cd tsp-visualizer/frontend
python -m http.server 8080
```
Открыть в браузере: http://localhost:8080
