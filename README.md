#  TSP Visualizer

##  Описание проекта
**TSP Visualizer** — это интерактивное веб-приложение для наглядной демонстрации алгоритмов решения задачи коммивояжёра (Traveling Salesman Problem). 
##  Как работает
Пользователь может расставлять города на карте, выбирать один из трёх алгоритмов поиска маршрута и наблюдать за пошаговой анимацией его построения. 

##  Основные возможности
-  **Интерактивная карта** — добавление городов кликом мыши на поле Canvas.
-  **Три алгоритма поиска маршрута:**
  - *Жадный* — быстрое приближённое решение.
  - *Полный перебор* — точное решение для ≤ 10 городов.
  - *Муравьиный* — эвристический алгоритм с балансом скорости и качества.
-  **Пошаговая анимация** — визуализация процесса построения маршрута.
-  **Статистика в реальном времени** — длина маршрута, время выполнения алгоритма, количество городов.
-  **Лог анимации** — отображение каждого шага построения маршрута.
-  **Адаптивный дизайн** — корректное отображение на всех устройствах.
